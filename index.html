git for dummies

Installer Git

  1. Download et git program, f.eks. https://git-scm.com/downloads
  2. Installer programmet
  3. Højreklik i en folder, og vælg "Git bash here"
  4. Du kan nu køre git i en folder

Git init

  1. Åben "git bash" i den folder du gerne vil køre git i.
  2. Du kan også navigere rundt ved at bruge "cd (foldernavn), eller "cd ../" for at gå en op.
  3. Når du er i din ønskede folder, skriv da kommandoen "git init"
  4. Husk "git" foran alle git-kommandoer

Git status

  1. "git status" giver dig informationer om den folder du er i.
  2. Hvis navnet er rødt, er det ændret arbejde, som ikke er gemt endnu.
  3. Hvis det er grønt, er det staget og klar til commit.
  4. Den fortæller også om hvilken branch du er i, og og den branch er up-to-date.

Git add

  1. "git add (navn)" stager dine filer, klar til commit.
  2. Du kan også skrive en folder, så tager den alt indhold i den.
  3. Du kan også få den til at tilføje alt med "git add ."

Git commit

  1. "git commit" gør dine filer klar til at blive sendt til dit repository.
  2. Du giver dit commit en titel
    med " -m 'titel' "
  3. Det er en god ide at bruge "git status" en gang imellem, for at sikre sig ens kommando kom igennem

Git push

  1. "git push" sender dine filer til git-serveren efter du har commited
  2. Du burde nu kunne se dine filer i dit repository

Git branch

  1. "git branch" er lidt som status, fordi den viser hvilken branch du er på i øjeblikket.
  2. Dine filer bliver lagt op på den branch som du er på
  3. Man har altid en "master" branch, som er standarten for dine commits

Git checkout

  1. "git checkout" er en kommando til at skifte hvilken branch man arbejder på
  2. Man kan altid checke hvilken branch man er på med "git branch"

Git merge

  1. "git merge" bruges til at få dit arbejde fra en branch til e
